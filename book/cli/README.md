# CLI User Guide

The CREATOR Command-Line Interface (CLI) provides a powerful text-based environment for assembly programming, debugging, and testing. It's designed for advanced users, automation, and environments without graphical interfaces.

![CLI Main](./img/cli_main.png)
*Figure: CREATOR CLI interactive prompt.*

## Overview

The CLI version offers several advantages over the web version:

### Key Features

- **Interactive Mode**: Full-featured REPL with command history
- **Powerful Debugging**: Step, unstep (reverse), breakpoints, and stack tracing
- **State Management**: Save and restore complete simulator snapshots
- **Configuration System**: Customize aliases, shortcuts, and preferences via YAML
- **Tutorial Mode**: Built-in interactive tutorial for learning RISC-V
- **Automation**: Script execution and batch processing
- **Accessible Mode**: Screen reader support for visually impaired users

### Use Cases

1. **Learning**: Use tutorial mode to learn assembly programming interactively
2. **Development**: Write and test programs with powerful debugging tools
3. **Automation**: Script testing and validation workflows
4. **Remote Work**: SSH-friendly text-based interface
5. **CI/CD**: Integrate assembly testing into pipelines
6. **Accessibility**: Use with screen readers for inclusive development
